## Store V2 learning routes

This project now includes a second learning area named Store V2.

- /store-v2: Catalog, reusable components, custom directive, custom pipe
- /store-v2/product/:id: Route parameter + product details page
- /store-v2/cart: Reactive form checkout and validation
- /store-v2/playground: Template-driven form, ngSwitch, and json pipe

Store V2 is heavily commented to help beginners learn Angular v15 concepts step by step.
